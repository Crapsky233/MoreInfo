This mod can display a lot of info in the game, all info display can be turned On/Off in Mod Configuration.
Whether you are a player or a modder, there is always something for this mod to help you.
The mod is client-only, which means you can use it in a server and the server does not need to enable this mod.
The functions of this mod include but are not limited to:

Display chest name and capacity - to make your sorting more convenient. (TODO: supporting Magic Storage Mod)
Display Player Luck - control over Luck system. (It does CONFLICT with Max Luck Mod since that Mod Hooked Player.RollLuck to "modify luck".)
Display the biome you are in (support modding environment) - help you build facilities such as monster farms better.
Turn off favourited item tooltip - because why the heck not?
Accessories in vanity slot display tooltips as usual - view accessories' tooltips conveniently.
Display mouse pointer/player coordinates - easier to debug mods
Display various info about items/walls/tiles - easier to debug mods

If you have good suggestions for this mod, such as adding information you want or improving displayments, you can leave a comment on the mod's Steam Workshop page.

这个模组可以显示游戏中的诸多信息，一切信息显示可在模组配置面板中开启或关闭.

显示鼠标指针/玩家坐标 - 调试模组更加轻松.
显示物品/背景墙/物块的各种信息 - 方便调试模组.
